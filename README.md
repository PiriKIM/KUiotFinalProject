## 📌 프로젝트 개요
자세요정 (자세를 잡아주는 자세 교정 메이트)

### 🎯 목표
- 장시간 앉아 있는 현대인의 생활 습관으로 인한 자세 불균형 및 만성 통증 문제를 해결하기 위해, 웹 기반의 실시간 자세 분석 및 교정 피드백 시스템을 구현하고자 합니다.
- 사용자는 웹캠만으로 실시간 자세 분석이 가능하며, 잘못된 자세가 감지되면 시각적/음성 피드백을 통해 교정 가이드를 제공받습니다

### 💡 주요 기능
- OpenPose 또는 MediaPipe를 활용한 관절 좌표 추출

- Flask 또는 FastAPI 기반의 API 서버

- Python, OpenCV, Pandas 활용한 데이터 처리

- CSV 또는 SQLite를 이용한 로컬 저장소 구성

- Flask/HTML/CSS/JavaScript로 웹 인터페이스 구성
